# IRONFALL
Juego IRONFALL

## Descripcion breve del juego: 
El juego trata de conquistar todo el continente de Feronis, derrotando a distintas facciones y consiguiendo mejoras a lo largo del juego.

## Tecnologias principales y Plataformas:

-Software TILERD
-lenguaje: Java
-IDE: IntelliJ IDEA
-PC (Windows)
